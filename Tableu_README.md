# Tableau Dashboards

## Overview

This section presents four interactive Tableau dashboards developed to support data analysis, model evaluation, and business insights. These dashboards transform processed data and machine learning outputs into meaningful visual representations for better understanding and decision-making.

## Dashboards Included

### 1. Data Quality Check

This dashboard focuses on assessing the quality of the dataset. It highlights missing values, inconsistencies, and distribution of key attributes to ensure the dataset is clean and suitable for further analysis and modeling.

### 2. Model Performance

This dashboard visualizes the performance of machine learning models using evaluation metrics such as Accuracy, Precision, Recall, and F1 Score. It enables comparison between models to identify the most effective approach.

### 3. Business Insights

This dashboard provides meaningful insights derived from the dataset. It helps in understanding patterns, trends, and relationships that can support data-driven decision-making from a business perspective.

### 4. Scalability Runtime

This dashboard analyzes how the system performs with increasing data size. It focuses on execution time and scalability, demonstrating the efficiency of the implemented solution in handling larger datasets.

## Tools Used

* Tableau Public / Tableau Desktop
* PySpark processed dataset
* Machine learning model outputs

## Features

* Interactive dashboards with filters and parameters
* Comparative analysis of model performance
* Clear visualization of data quality and trends
* Scalable performance insights

## How to Use

1. Open the `.twbx` files using Tableau Desktop or Tableau Public.
2. Navigate through dashboards using tabs.
3. Apply filters to explore different insights dynamically.
